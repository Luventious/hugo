## 背景

新能源汽车发展时至今日，很多车企的展车模式已经形成本品牌的特色，但对于仍未找准切入点的部分合资车企而言，店端的车辆展示存在关键短板，核心问题聚焦于**数字化、智能化及科技化配置的体验传达不足**：展车未能通过直观、可交互的方式呈现产品核心科技亮点，导致客户无法快速感知其智能优势，不仅直接影响线下看车的体验感与满意度，更使得与新势力品牌、主流新能源车型在客户体验层面的差距持续拉大 —— 竞品普遍以场景化、沉浸式的科技体验设计打动消费者



## 挑战


## 行业趋势

相较于传统以静态空间与造型展示为主的展车模式，当前品牌展车正逐步强化对产品核心价值与使用场景的主动化呈现能力。该转变使得展车体验从“感知外观属性”延伸至“理解产品能力与使用方式”。在此背景下，各品牌之间的展车体验差距不再主要源于硬件造型差异，而是集中体现于数字化功能的丰富程度及其可感知性，以及座舱系统所营造的沉浸式交互体验水平。




## 走访4S店

本次调研聚焦于对 4S 店潜在用户看车过程的实地走访，旨在达成以下关键目标

**精准捕捉用户行为、收集专业见解、洞察体验核心、提炼典型旅程与痛点。**




## 核心洞察


HMW QUETIONS：我们如何通过为用户提供**沉浸式与场景化的静态展示，**从而**增强客户与店端的信任关系，**以传达品牌**智能互联/操控性能的体验价值？**




## 体验策略

客户与销售的信任关系建立难：

| **洞察** | **策略** |
| ---------- | ---------- |
| 客户与销售的信任关系建立难 | 构建“透明化+专业化+情感化”三维信任体系 |
| 配置与价格信息传达低效 | 打造“分层化+可视化+互动化”信息传递体系 |
| 数字化座舱感知弱 | 构建“场景化+沉浸式+个性化”座舱体验感知体系 |
| 试驾浅尝辄止 | 打造“全场景+定制化+深度化”试驾体验体系 |
| 静态体验保守沉闷 | 构建“互动化+场景化+主题化”静态体验体系 |
| 智驾功能认知 / 接受度低 | 构建“阶梯式+可视化+场景化”智驾认知普及体系 |

> 用户视角原则：所有体验设计均以用户需求为核心，避免“技术导向”，用用户易懂的语言、可感知的场景传递产品价值。

## 设计落地

后续需依托设计落地方案推进实践，并遵循迭代优化原则，结合用户反馈持续完善，真正实现展车体验从“静态陈列”到“价值传递”的转型，助力合资车企在新能源市场竞争中重塑体验优势。
